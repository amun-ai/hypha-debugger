Hypha Debugger

Drag the button above to your bookmarks bar. Then click it on any page to inject the debugger.

How it works

  1. Click the bookmarklet on any web page
  2. A floating bug icon appears — click it to see the service URL
  3. Paste the URL into your AI agent (Claude, GPT, etc.) to let it control the page
  4. The agent can inspect DOM, click elements, fill forms, take screenshots, and more

Features

Auto-reconnect — debugger survives page reloads and same-origin navigation
Link interception — clicking links keeps the debugger alive
Smart DOM — indexed interactive elements [0], [1], [2]...
AI cursor — animated cursor shows what the agent is doing
React support — inspect React component tree via fibers
Screenshots — capture page state as PNG

Manual install (copy the code)

Create a new bookmark and paste this as the URL:
